## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   git clone https://github.com/your-username/portfolio.git

2. Install dependencies
   npm install

3. Start the development server
   npm run dev

4. Build for production
   npm run build
